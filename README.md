# 自炊save


## サービス概要
「自炊って本当にお得なの？」の答えを数字で教えてくれるアプリです。
料理記録から外食との差額を自動計算し、節約効果を可視化します。
成功体験の積み重ねで、自炊が楽しく続けられるようになります。


## このサービスへの思い・作りたい理由
世の中、節約レシピは溢れているのに、「本当に節約できてるの？」の答えは誰も教えてくれません。
この **「見えない成果」** が、自炊継続の最大の障壁だと考えています。

私自身、自炊をしていてもコンビニご飯とどれくらい金額差があるのか分からず、継続のモチベーションを失ってしまうことがありました。

そこで、自炊の最大のメリットである **節約額** にフォーカスし、
金額の差を手軽に可視化することで、自炊を前向きに続けられるサービスを作りたいと思いました。
「なんとなく節約できてはいるんだろうな、、」という考えから「約〇〇円節約！」と変換されるだけで
自炊に対しての見え方がポジティブなものになり、継続化や自炊を始めるきっかけにもなると考えています。

レシピアプリ → 作り方は分かるが節約効果は不明
家計簿アプリ → 支出は記録できるが外食との比較はできない
類似アプリでも解決できなかった  
**「自炊するとどれくらい得をしているのか」**  
このシンプルな疑問に応えるために、
**自炊 × 節約**
という新しい切り口で、自炊のモチベーションをあげていきたいと考えました。

自炊の習慣化への課題は、LINE通知の機能を付け、ユーザー自身のペースで設定することで、
「毎日は大変」「お昼のお弁当だけでも、、」とユーザーの生活に見合った自炊に無理なく取り組むことができると考えています。


## ユーザー層について
メインの年代：20代〜30代
ターゲット：一人暮らしの方・節約したい社会人・自炊にハードルを感じている方
理由：自炊に興味を持つ一人暮らしの方や、お昼のお弁当だけでも節約したいという社会人。
そんな方こそ**節約額**というメリットを感じることで、自炊に対して前向きに頑張ってみようと思えるサービスだと思います。


## サービスの利用イメージ
1.**料理記録**: 作った料理名・使用食材・食材のグラム数を登録
   → システムが自動で材料費を計算
2.**節約効果の可視化**: 「コンビニご飯と比べて約150円安い！」
   → 即座に成果が数値で分かる
3.**継続サポート**: LINE通知で「今日も自炊頑張りましょう！」
   → ユーザーのペースに合わせた習慣化支援
### その他
- プロフィール画面に、「節約金額」「節約金額の日別グラフ」「先月との節約金額比較」「目標」「連続自炊記録」などの成果を表示
- ユーザーが作成したレシピ(MYレシピ・非公開)を公開レシピへ投稿可能


## ユーザーの獲得について
- Xシェアでの目標達成や節約金額共有（サービス拡散）
- ゲストログインを導入し、レシピ閲覧のみ可能（雰囲気を知り、使用ハードルを下げる）


## サービスの差別化ポイント・推しポイント
### 差別化ポイント
サービスの軸としては**自炊によるメリットの実感**。
その手段として**節約金額**を可視化。

| サービス | 目的 | 主な機能 | 得られる体験 |
| ---- | ---- | ---- | ---- |
| レシピアプリA | 調理をスムーズにする | レシピ検索・調理手順 | 「作り方」がすぐ分かり、迷わず料理できる |
| 家計簿アプリA | 支出管理を最適化する | クレカ連携・自動記録 | お金の流れが自動で見える化される |
| 自炊save | 自炊の節約効果を見える化する | 自炊の記録・外食と差額可視化 | 「自炊した分だけ節約できた」が実感できる |

### 推しポイント
- 節約金額は、「コンビニご飯 - 自炊料理 = 節約金額」で算出。
- 料理の記録・金額の計算を、食材と使用グラム数入力で、開発者が用意したマスターデータを元に計算を簡易化or手動で細かく設定できます。
- LINE Messaging APIを使用し、LINEでの通知で自炊の習慣化を促せます。

## 機能候補
### MVP

- ログイン機能（googleアカウント）
- マスターデータ（計算の元となるデフォルトのレシピと金額）
- レシピ一覧（MYレシピ(非公開)・ユーザー公開レシピ）
- 自炊記録機能・フォーム(MYレシピに追加・登録時にマスターデータを元に計算・ユーザー手動手動)
- プロフィールページ（節約金額・節約金額グラフ・目標設定）
- グラフ表示（月間節約額の推移など・chart.js）

### MVPリリース後追加

- 目標達成でメッセージをポップアップ表示
- プロフィールに記録追加(目標達成数・連続自炊日数・1ヶ月あたりの最高節約金額)
- 検索機能
- ログイン方法追加（LINE）
- LINE連携でリマインド通知追加
- X共有
